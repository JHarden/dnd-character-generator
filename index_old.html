About Me

Hi, I'm John. I live in Vancouver and work as a web developer at PNI Digital Media. I've been in Vancouver for over 2 years and I love it here. I used to work for Central 1 as a Software Specialist 2 and I'm originally from Ireland, where I worked as a Java Developer for 3 years. I'm much more interested in front-end development and design though, and I'm always looking to learn the latest tech and trends in the industry.

In my current position I make use of a lot of cool technologies like Rect, Redux, Angular 1.5, SASS, PostCSS, jQuery and plain ol' javaScript with task runners such as gulp and grunt. I've also used webpack as a task runner/bundling tool. I tend to use VS Code or WebStorm as my IDE of choice, with Visual Studio for any back end changes.

I enjoy running, camping and travelling, skiing and playing board games, the occasional social drink and exploring this beautiful country.
